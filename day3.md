# Maratona Business English

## Preposições

```
Prepared BY: preparado por alguém
Prepared FOR: preparado para alguém ou para alguma situação (for + subs. ou pron.)
Prepared TO: preparado para alguém ou para alguma situação (for + verb.)
```

```
Given BY: dada por alguém
Given TO: dada para alguém
Given FOR: dada para alguém no lugar de alguém
	   Ex: Give John a message for me.
Given FOR: no sentido de daria ou pagaria por algo
	   Ex: How much do you give for this house?
```

```
Mad at/with: bravo com alguém.
	     Don't be mad at John.
Mad about: amar ou adorar algo ou alguém.
	   Ex: I am mad about him.
```

```
Increase/reduce BY: aumentar ou reduzir EM
		    Ex: We reduce our costs by 30\%.
Increase/reduce TO: aumentar ou reduzir PARA
		    Ex: Our revenue increased to 4MM.
Increase/reduce OF: aumentar ou reduzir DE
		    Ex: We had a cost reduction of 300K.
```

```
Get back TO: retornar alguém (telefone, e-mail...)
	     Ex: Can I get back to you later?
Get back at: se vingar de alguém
	     Ex: Don't mess with John, he will get back at you later.
```

```
Made OF: feito de - material
	 Ex: Made of cotton.
Made WITH: feito com - ingredientes, produtos
	   Ex: Made with eggs.
Made IN: origem
	 Ex: Made in France.
Made BY: feito por uma pessoa ou empresa
	 Ex: Made by me. Made by Tesla.
Made FOR: feito para alguém
	  Ex: Made for you.
Made TO: feito para um propósito
	 Ex: Made to last.
```

```
DEPOIS DA PREPOSIÇÃO - VERBO + ING

Ex: She is responsible FOR developING people.
    Thank you FOR sharING the information.

PREPOSIÇÃO TO:
Exige verbo no ING depois: look forward to, used to, prior to.
```

## Prefixos/Sufixos

```
UNDER: A MAIS - UNDERESTIMATE, UNDERCOOKED
OVER: A MENOS - OVERESTIMATE, OVERCOOKED

Underpromise and overdelivery
```

```
NESS: transforma adjetivo em substantivo
Bright: Brightness
Happy: Happiness
```

```
ABLE: associado a um verbo para demonstrar uma ação possível de ser executada.
Understand: understandable
Measure: mesurable
Accept: acceptable
Convert: ConvertIBLE
```

```
Dismiss, fire: demitir - pode ter culpa da demissão
Let go, lay off, make someone redundant - situação foge ao desempenho do funcionário

Resign, quit: quando o funcionário pede a conta
```

```
Lose: perder competição, jogo, posse, estar perdido
Miss: perdas que não envolvem posse
      Ex: Miss the bus.
```
